# POS .NET8

## ขั้นตอนการติดตั้ง

1. ติดตั้ง Docker
   - ดาวน์โหลดและติดตั้ง Docker Desktop จาก https://www.docker.com/get-started
   - รีสตาร์ทเครื่องถ้าจำเป็น
   - ตรวจสอบว่า Docker ทำงานปกติด้วยคำสั่ง:
     ```bash
     docker --version
     ```

2. เตรียมโปรเจค
   - ตรวจสอบว่าไฟล์ `Dockerfile` และ `docker-compose.yml` อยู่ในโฟลเดอร์หลักของโปรเจค

3. สร้างและรันคอนเทนเนอร์
   - เปิดเทอร์มินัลหรือ PowerShell ในโฟลเดอร์โปรเจค
   - รันคำสั่ง:
     ```bash
     docker compose up --build
     ```

4. ตรวจสอบการทำงาน
   - รอจนกว่าคอนเทนเนอร์จะสร้างเสร็จ
   - เปิดเว็บเบราว์เซอร์ไปที่ `http://localhost:5000` หรือพอร์ตที่กำหนดในโปรเจค

5. ปิดและลบคอนเทนเนอร์
   - เมื่อต้องการหยุดให้กด `Ctrl+C` ในเทอร์มินัล
   - หรือรันคำสั่ง:
     ```bash
     docker compose down
     ```

6. ข้อควรระวัง
   - ตรวจสอบเวอร์ชัน Docker ให้เข้ากันกับ .NET 8
   - หากมีการใช้ฐานข้อมูล ให้ตรวจสอบการตั้งค่า connection string ในไฟล์คอนฟิกของโปรเจค
